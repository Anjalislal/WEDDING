# Wedding Invitation Website

Open `index.html` in a browser to view the invitation.

Edit all wedding text, links, dates, photos, and music in `config.js`.

## Asset Names

Place your files here, or update the paths in `config.js`:

- `assets/photos/cover.jpg`
- `assets/photos/couple-hero.jpg`
- `assets/photos/couple-portrait.jpg`
- `assets/photos/gallery-1.jpg`
- `assets/photos/gallery-2.jpg`
- `assets/photos/gallery-3.jpg`
- `assets/photos/gallery-4.jpg`
- `assets/photos/gallery-5.jpg`
- `assets/photos/gallery-6.jpg`
- `assets/photos/preview.jpg`
- `assets/music/background.mp3`

For WhatsApp sharing, upload the site to a public host and keep `preview.jpg` as a clear couple/photo preview image.
